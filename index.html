Instructions

  1. Find your current save code by going to: Settings () > Export and copy the code.
  2. Paste the save code and enter what you'd like to change above.
  3. Click "Update Save Code" to generate the updated codes.
  4. Copy the updated code using the "Copy New Save Code" button above.
  5. Import the new code by going to: Settings () > Import.
  6. (EXTRA) If you are using the non base64 encoded code, you need to make sure you encode it using a base64 encoder first!
